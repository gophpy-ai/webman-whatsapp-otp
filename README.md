# Webman WhatsApp OTP Plugin

A complete OTP login plugin for **Webman**, supporting:

✔ WhatsApp Cloud API  
✔ OTP login  
✔ JWT Access Token  
✔ Refresh Token  
✔ Middleware authentication  
✔ Users migration  
✔ Composer installation  

## Installation

```bash
composer require gophpy-ai/webman-whatsapp-otp
```

## Features
- Send OTP via WhatsApp Cloud API
- Verify OTP + auto-create users
- Issue access/refresh tokens
- Token refresh endpoint
- Middleware for protecting routes
- SQL migration included
